- Playback range start/end controls.

5) KNOWN ISSUES / RISKS TO CHECK FIRST
==================================================
1. DPI/layout clipping is improved but not fully closed
- Must still test at 100/125/150% and narrow window widths.
- Focus checks: timeline control row, palette tabs, layer action rows.
- Recent pass added DPI-scaled sizing for main rows/columns and panels; verify no regressions at 1120px+ widths.

2. Undo/redo is snapshot-based, not command-granular
- Works functionally but can be memory-heavy on large docs.
- Stroke grouping and operation-specific history compression are not done.

3. Selection transform depth is partial
- 90 CW and 90 CCW now exist.
- No scale/skew/rotate interactive handles yet.

4. Playback UX semantics can still be confusing
- Playback has two modes of sequencing (selected frames vs range).
- Consider explicit UI indicator of active playback source.

5. Palette management is still basic
- No palette preset import/export yet.
- No palette lock mode.
